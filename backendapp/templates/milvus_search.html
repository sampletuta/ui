How to use Face Search:
  1. Upload an image containing one or more clear faces
  2. Set the number of results you want (1-20)
  3. Adjust the confidence threshold (0.0-1.0)
  4. Click "Search for Similar Faces"
  5. View the top matching faces from your watchlist

Note: Higher confidence thresholds will return more similar matches, while lower thresholds will return more potential matches.
Multiple Faces: If your image contains multiple faces, the system will search for each detected face separately and provide combined results.
